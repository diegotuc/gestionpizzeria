Arquitectura del proyecto

Frontend → HTML + CSS + JS (venta.html, styles.css, caja.js)
Backend → Node.js + Express (server.js(rutas /caja))
Base de datos → maneja entidad Caja y movimientos

Frontend funcional con venta.html, caja.js y styles.css
